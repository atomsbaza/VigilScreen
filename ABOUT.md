## Full Description (for README intro)

### One-liner
**DockLock** is a privacy-first macOS Menu Bar app that auto-locks your Mac when you walk away and hides sensitive data with a single keystroke.

### The Problem
You're working on something sensitive — source code, credentials, API keys, documents.
Your phone buzzes. You get up for 30 seconds.
**Someone could see your screen.**

### The Solution
- **Proximity Lock**: Phone goes out of range? Mac locks automatically
- **Panic Mode**: One hotkey (⌘+Shift+L) hides your Terminal, IDE, browser, Slack instantly
- **Local-first**: Everything stays on your Mac. No cloud. No tracking

### Why DockLock?
✅ **For anyone with sensitive data** — developers, consultants, security professionals
✅ **Open source** — community audits the code for security
✅ **Zero dependencies** — pure Apple frameworks, no third-party trust required
✅ **macOS native** — SwiftUI + AppKit, feels like a Mac app should

## Features Highlight (for README section)

### 🔒 Proximity Lock
- Auto-lock when iPhone/Watch goes out of range
- Smart hysteresis to prevent false triggers
- Works with Bluetooth or WiFi RSSI

### 🚨 Panic Mode
- Single hotkey (customizable) instantly hides your apps
- Configurable blocklist (Terminal, IDE, browsers, etc.)
- Release with Touch ID for added security

### 🔐 Security First
- No network calls, no cloud sync (MVP)
- Entitlements disclosed openly in code
- Responsible disclosure policy for vulnerabilities

### 🏗️ Built with Modern Swift
- SwiftUI 6 + AppKit
- Swift 6 concurrency (async/await)
- macOS 15 Sequoia+
